        1. Écrire une fonction ```suivant(n)``` qui renvoie le successeur du nombre ```n```, suivant les règles énoncées ci-dessus.

    === "Correction"
        {{ correction(True,
        "
        1.
        ```python linenums='1'
        def suivant(n):
            if n % 2 == 0:
                return n // 2
            else:
                return 3*n + 1
        ```
        2.
        ```python linenums='1'
        def syracuse(n):
            print(n)
            while n != 1:
                n = suivant(n)
                print(n)
        ```
